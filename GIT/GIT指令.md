### 基本命名
git clone url 克隆远程仓库的代码
git status  查看当前状态（分支）
git add .  将被修改的文件从工作区放到暂存区
git commit -m'注释'    将文件从暂存区放到本地仓库
git push origin master  将本地仓库里的文件提取到远程仓库
