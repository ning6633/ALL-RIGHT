# css语法

css（Cascading  style  sheet）汉译为  **层叠样式表**  是用于控制网页样式

语法组成：
			选择符（对标签进行选取）和   声明
					声明分为**属性和属性值**

​		语法：选择符{
​									属性1：属性值；
​									属性2：属性值；
​									属性3： 属性值1 **空格**  属性值2；}	
​				如果一个属性有多个属性值，**属性值与属性值之间用空格隔开**

background 背景颜色

​		所有的CSS语句都要放在CSS样式表里面<style></style>

​			小明      穿衣服      红色的衣服     

< div>< /div>:    把网页分块儿  换行

< span>< /span>  ：  不换行
